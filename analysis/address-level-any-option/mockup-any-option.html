Design notes

"Any" appears at the top of each non-lowest level (Block, Village), separated from the specific options by a thin divider.
Teal outline radio distinguishes "Any" visually. A subtle caption ("Include all blocks") clarifies intent.
HWC level has no "Any" — it is the lowest level in the hierarchy, so the option is not shown.
Selecting "Any" at Block would deselect all specific blocks and pass a wildcard to the query — the Village list would still appear below.
